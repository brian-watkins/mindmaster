# MindMaster

A code breaking game that is an excuse to try out agile architecture in Elm.


### Setup

- Install node
- `$ npm install`

### Run the Tests

To run all tests:

```
$ npm test
```

To run the elm tests:

```
$ npm run test:elm
```

To run the integration tests:

```
$ npm run test:integration
```

### Running Locally

```
$ npm run dev
```

Then visit http://localhost:3000 in your browser.
